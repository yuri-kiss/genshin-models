# genshin-models

This is just a repo of the genshin models if the hosting site ever dies.

It uses my modified google document (in CSV format) to download all the files.

https://docs.google.com/spreadsheets/d/1KWw_FFQJyWj7dMYpca1MHtNdufqoKelbJhB0om7uvzE/edit?usp=sharing

## CLI Usage

Just run `npm ci` (node v22)

and then you can use `node . --help` to see help on using the CLI.
